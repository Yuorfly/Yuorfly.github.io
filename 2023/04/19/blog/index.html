基于hexo和github搭建blog的操作步骤

1、git下载
git官网地址：https://git-scm.com/downloads （因为是外网，网速比较慢）
国内镜像地址：https://registry.npmmirror.com/binary.html?path=git-for-windows/（阿里镜像地址，一般版本会比官网低一点）

2、git安装
选择安装地址

然后一路默认就可以

git bash here和git gui here这两个功能选中后添加到右键菜单栏中！

然后一步步点击下一步，进行安装就可以了！

3、hexo的搭建
1、node的安装
下载地址：https://nodejs.org/zh-cn/download/

选择这个，在下载好之后，直接双击打开，然后一步步安装就好！
2、hexo的下载
选择一个文件夹，打开 bash
输入hexo的安装命令！

npm install -g hexo 安装hexo命令
hexo -v 查看hexo是否安装好

  • 这样安装环境就已经搭建完成，接下来就可以开始搭建网站啦！

3、链接到github

设置连接到用户名：git config --global user.name "GitHub 用户名"
设置连接用户邮箱：git config --global user.email "GitHub 邮箱"
设置好之后，为了确定有没有设置好，不妨查看一下
查看用户名：git config user.name
查看邮箱：git config user.email

创建密钥:ssh-keygen -t rsa -C "这是你的邮箱"

点击enter后显示如图表示创建完成
打开C:\Users\yuhuo\.ssh（被隐藏了，需要打开隐藏项目）
用记事本打开id_rsa.pub并复制其中内容

打开github，添加密钥

验证是否连接成功

ssh -T github@github.com


显示连接成功

4、创建github仓库

创建的名字需要使用 用户名.github.io
然后打开仓库，复制仓库名
博客的默认地址为：https://用户名.github.io
5、hexo初始化以及本地预览

hexo init #初始化
hexo install #安装组件

我已经有了仓库，并且配置好了，目前只是在另一台电脑对git进行配置，方便可以对界面进行优化，所以不建议采用初始化操作！！！

hexo g #生成界面
hexo s # 启动预览

6、网站部署到GitHub仓库

npm install hexo-deployer-git –save #部署博客插件

然后修改_config.yml

hexo d 完成部署到github仓库中
打开github检查仓库有没有上传上去，网络波动有点大，多上传几次!!!!
